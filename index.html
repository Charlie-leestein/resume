Program

  • Face Recognition-tensorflow

    An efficient face alignment system based on explicit 3D face modeling; a CNN architecture with a locally connected layer, which, unlike regular convolutional layers, learns different features from each region in the image. At the same time, the system achieves similar results by training 60 different CNNs on patches containing ten regions, three scales, and RGB or grayscale channels. During the test phase, 160 bottleneck features are extracted from each patch, plus its horizontal flip, to form a 19200-dimensional feature vector (160×2×60). The proposed CNN architecture also uses a locally connected layer. Its verification results are obtained by training a joint Bayesian classifier on the 19200-dimensional feature vector extracted by this CNN.

  • Asset Recognition

    Asset identification is divided into three parts:

    1) Clustering: First, the kmeans algorithm with adaptive K value is used for preliminary clustering, and then the Simhash algorithm is used for secondary clustering. The final asset clustering accuracy reaches more than 95%.

    2) Recommended keywords: First, the TF-IDF algorithm is used to generate keywords, and smoothing technology is added. Then the public keywords generated by each asset are used as recommended keywords.

    3) Generate fingerprints: Select the number m of 1/n public keyword groups with the highest weight of a type of asset data (the size of m value is determined according to the specific scenario and effect, generally m=3) and design as follows:

    i) Combined fingerprint priority: key field>important field>remaining public keywords (descending order)

    ii) Combined fingerprint selection design: the weight of any selected keyword must reach the threshold

    iii) Adaptive threshold design: Use threshold adaptive method [the threshold is 1/n of the highest weight of each component keyword, the default n=2]
